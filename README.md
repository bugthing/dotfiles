My dot files
============

Here is a repo containing my dot files.

I use the gnu tool `stow` to manage them.

Install stow
------------

    $ pacman -S stow

New dir?
--------

    $ git clone git@github.com:bugthing/dotty.git ~/dotfiles

    $ stow bash
    $ stow vim
    $ stow tmux
    $ stow desktop
    $ stow ruby
